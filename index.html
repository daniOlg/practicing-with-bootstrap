Sistema biblioteca virtual

Sistema para administrar libros solicitados por usuarios inscritos en nuestro sistema, incluye sistema de busqueda de libros por palabras clave, aspectos tecnicos: funciona con localStorage, implementa un registro de lectores y para luego solicitar el libro, para comenzar puede inscribir un lector
